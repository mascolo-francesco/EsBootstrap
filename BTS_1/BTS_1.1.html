Esercizio BTS_1.1 – Introduzione a Bootstrap 5

1. A cosa serve Bootstrap?

Bootstrap serve per creare siti web moderni e responsive in modo rapido, utilizzando un insieme di classi CSS e componenti già pronti. Permette di costruire layout ordinati e adattabili a ogni tipo di schermo (PC, tablet, smartphone) senza scrivere tutto il codice CSS da zero.

2. Cosa devi importare per creare una pagina web con Bootstrap?

Bisogna importare il file CSS e gli script JavaScript di Bootstrap, di solito tramite CDN:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script>

3. Cos’è un CDN? Che vantaggi comporta?

Un CDN (Content Delivery Network) è una rete di server distribuiti nel mondo che fornisce i file (come Bootstrap) da un server vicino all’utente. I vantaggi sono:

  • Caricamento più veloce delle pagine.
  • Minor utilizzo di banda dal tuo server.
  • File sempre aggiornati.

4. A cosa serve questo attributo del tag meta?

content="width=device-width, initial-scale=1"

Serve per dire al browser di adattare la larghezza della pagina alla larghezza dello schermo del dispositivo, mantenendo una scala di zoom iniziale di 1. È ciò che rende la pagina responsive.

5. Che differenza c’è tra container e container-fluid?

  • container: ha una larghezza massima che cambia in base alla dimensione dello schermo (centrato e con margini ai lati).
  • container-fluid: occupa sempre il 100% della larghezza della finestra, indipendentemente dalla dimensione dello schermo.
